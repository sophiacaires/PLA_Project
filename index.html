Criptografia e Descriptografia de Mensagens

Siga estas diretrizes de formatação para garantir que sua chave seja interpretada corretamente pela aplicação:


Estrutura

- Utilize colchetes para iniciar e fechar a chave;
- Cada linha deve estar entre seus próprios colchetes;
- Elementos dentro de uma linha são separados por vírgulas;
- Separe cada linha por vírgulas.


Exemplo de chave correta:

[[2, 17], [3, 11], [5, 13]]


Observações adicionais

- Os elementos da matriz podem ser números inteiros ou decimais.
- Utilize o ponto "." para separar a parte decimal de um número.
- Evite espaços ou outros caracteres dentro da matriz.
- As vírgulas servem apenas para separar elementos dentro de uma linha e as próprias linhas.
